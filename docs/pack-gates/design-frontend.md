# design-frontend Gate

Design read: developer authority portfolio for hiring managers and technical security leaders, with a clean technical dossier language.

Applied decisions:

- Avoided AI-purple gradients, stock hacker visuals, glass-heavy cyber cliches, and generic three-card hero treatment.
- Used a dossier/map first viewport with explicit investigation system language.
- Chose restrained beige/teal/black technical palette, mono labels, hard borders, and diagram-forward case pages.
- Added responsive constraints for cards, navigation, diagrams, and timeline.

Verification target:

- `npm run lint`
- `npm run build`
- `npm run qa:browser`
