Security agents need context that is scoped, current, and inspectable. MCP helps by turning local tools and knowledge into explicit resources instead of invisible prompt ballast.

For cyber workflows, the important design questions are operational: which resources are authoritative, which are generated, which are stale, and which should never be exposed to the model.
