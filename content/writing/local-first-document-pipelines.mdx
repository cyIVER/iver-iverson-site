deck: "Document conversion for LLM workflows should default to local processing and explicit remote opt-in."

Document conversion looks like a convenience feature until the inputs are evidence-adjacent, sensitive, or proprietary. Local-first defaults turn the pipeline into a control: users can inspect what is extracted before deciding whether any remote enhancement is appropriate.
